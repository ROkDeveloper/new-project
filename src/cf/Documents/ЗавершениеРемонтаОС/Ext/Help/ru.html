Документ "Завершение ремонта ОС"

Документ предназначен для фиксации факта завершения ремонта основных средств, затраты на который в бухгалтерском учете учитываются, как отдельный объект основных средств (капитализируются).

При вводе документа в шапке нужно указать следующие реквизиты:

  • Подразделение - подразделение организации, которое выполняло ремонт. 
  • Вид ремонта - вид произведенного ремонта, капитализируемого в бухгалтерском учете.
  • Событие - наименование события в жизни основного средства, которое отражается документом.
  • Остались материалы - устанавливается, если после произведенного ремонта остались материалы для оприходования.
  • Склад - склад для оприходования оставшихся материалов.


В табличном поле Основные средства указывается список отремонтированных основных средств, а также плановый срок до следующего ремонта.
Также для каждого основного средства указываются материалы, оставшиеся после ремонта.
