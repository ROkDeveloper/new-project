Документ "Условия оплаты поступления"

ОС и НМА ® Поступление основных средств ® Условия оплаты поступления

Документ предназначен для изменения условий оплаты, если поступление оплачивается на условиях рассрочки платежа на период, превышающий 12 месяцев, или установленный организацией меньший срок.

При вводе документа в шапке нужно заполнить График платежей по договору. Общая сумма платежей по графику должна совпадать с общей суммой документа поступления.

Оценка в БУ - способ оценки рассрочки в бухгалтерском учете:

  • для способа Рассчитывается по ставке укажите ставку дисконтирования
  • для способа По цене без рассрочки в табличной части заполните колонку с ценой без рассрочки.

Цены в документе включают НДС (при наличии).


В табличной части (для спососба По цене без рассрочки ) указывается

  • Номенклатура - оборудование, товары или материалы. Недоступно для редактирования.
  • Цена по договору - цена из документа-поступления (с НДС, при наличии). Недоступна для редактироваия.
  • Цена без рассрочки - цена, которая была бы уплачена организацией при отсутствии рассрочки платежа. Указывать с НДС (при наличии).
